    Engine Info

    Chaos Chess currently exposes four computer engines plus an automatic mode. They do not all work the same way, and they do not all support the same rules.

    Auto Mode

    Picks the default engine for the selected ruleset. Classic games use Stockfish. Variant games use the custom Variant Search engine.

    Stockfish Strength: High

    A strong classic chess engine running in the browser. It only supports standard chess rules in this app. It does not understand friendly fire, kamikaze, wrap-around movement, or the custom pawn rules.

    Variant Search Strength: LOW

    A custom rule-aware search engine built on top of the Chaos Chess legality engine. It works across the variant combinations, but it is still early and much weaker than Stockfish on classic chess.

    Variant ML Hybrid Strength: MEDIUM

    A custom variant search engine with a learned value model blended into its evaluation. It is trained on Chaos Chess self-play and teacher-search data, and currently targets variant games rather than classic chess.

    Heuristic Baseline Strength: VERY LOW

    A simple evaluator with no deeper search. It is fast and useful as a baseline for experiments, but it is the weakest option strategically.

    Think Time

    Think Time controls the search budget for Stockfish and Variant Search. Higher values usually improve move quality but increase waiting time. The Heuristic Baseline mostly ignores this setting.

    Current limitations

    The ML-backed variant engine is still early-stage and tuned from relatively small self-play datasets. It has started to beat the plain variant search baseline in internal benchmarks, but it is still far less mature than Stockfish on classic chess.
